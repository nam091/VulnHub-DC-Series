# Cybersecurity Lab Series & VulnHub Write-ups

A comprehensive collection of professionally formatted penetration testing reports and capture-the-flag (CTF) write-ups. This repository documents various attack vectors, tools, and methodologies used across academic labs and real-world vulnerable machines.

## 📂 Repository Structure

The reports are organized into two main categories:

### 1. Core Lab Reports (`LabX.md`)
Detailed documentation of fundamental penetration testing concepts, focusing on:
- **Initial Access**: Password guessing, dictionary attacks, and password spraying.
- **Payloads & C2**: Utilizing frameworks like Metasploit, Sliver, and Empire.
- **Situational Awareness**: Host enumeration and internal network reconnaissance.
- **Privilege Escalation**: Bypass UAC, service exploitation, and token manipulation.

### 2. DC Series Write-ups (`DC-X.md`)
Full "Boot2Root" walk-throughs for the famous **DC Series** vulnerable machines from VulnHub:
- **DC-1 to DC-9**: Detailed exploitation paths covering Drupal CMS, WordPress, Linux privilege escalation (SUID, git, vi), and Kerberos attacks (Vé vàng/Golden Ticket).

## 🛠️ Tools & Technologies Used

These reports demonstrate proficiency in a wide range of industry-standard security tools:

- **Reconnaissance**: `nmap`, `netdiscover`, `cewl`, `dirb`.
- **Exploitation**: `Metasploit Framework`, `searchsploit`, `sqlmap`, `Impacket`.
- **Command & Control (C2)**: `Sliver`, `PowerShell Empire`, `Meterpreter`.
- **Credential Attacks**: `Hydra`, `John the Ripper`, `Hashcat`, `Responder`.
- **Post-Exploitation**: `PowerUp`, `SharpWMI`, `Mimikatz`, `SecretsDump`.
